## Description
This React project was used in order to collect data for a study conducted in the course "Current topic in interaction technology and design" at Ume√• University. The survey that was conducted needed users to be able to interact with UI-elements in both dark- and light-mode and then submit their preference.

## Libraries used
The projects uses a React front-end using UI-elements from Material-UI and animations from React Spring. The back-end is a MongoDB NoSQL-database connected via an Express middleware that uses a custom API for inserting the users survey answer to the database.
